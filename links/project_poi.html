Point of Interest Recommendation using Graph Neural Network

Tools:

This project focuses on developing a Graph Neural Network (GNN) to recommend Points of Interest (POIs) to users. The dataset contains user check-in data from Foursquare, representing interactions between users and POIs. To enhance the recommendation process, additional information about each POI, including categories, descriptions and photos, was extracted via the Foursquare API. These information were transformed into embeddings and then used to fed into the GNN to model relationships between users and POIs, generating personalized recommendations.
The architecture integrates different tools. ResNet and BERT models are used for feature extraction from visual and textual data of POIs.
The graph structure, where nodes represent users and POIs and edges represent check-ins, is implemented using the GATConv layer (Graph Attention Networks) in PyTorch Geometric. Additional libraries like Scikit-learn, Pandas and Numpy are used for data preprocessing and evaluation.
The model showed strong performance across several metrics, achieving high precision and recall, indicating its effectiveness in generating relevant recommendations. Additionally, recommendation accuracy metrics such as Precision@K, Recall@K and NDCG were employed, further validating the modelâ€™s capability.
However, the implementation faced challenges, particularly with the format of the Foursquare check-in dataset. Constructing the graph and generating negative edges in order to have a balance dataset was challenging. Despite these obstacles, the GNN effectively provided personalized and relevant POI recommendations for users.
