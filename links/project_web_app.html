Web Application for exchanging private messagges between users

Tools:

Together with two colleagues, we developed a web application for exchange messagges between users.
Specifically, the application allows each user to access their list of chats and interact individually with another user through message exchange. After the registration phase, various features were implemented: a section was created for adding or deleting a friend and displaying the friends list with access to their chats.
Regarding the technical architecture of the application, the backend was developed using Node.JS and Express.JS , ensuring efficient handling of requests, while React was used for the frontend. Additionally, Bootstrap was integrated for user interface design. Real-time interaction of messages has been achieved using socket.IO to ensure instant communication. To ensure authentication security, a hashing mechanism was used for login. For data storage, MongoDB was chosen, a flexible and scalable database that suits the needs of a messaging application well.
In conclusion, we successfully developed a web application with important features such as real-time messaging, friend management and a secure authentication mechanism, with a simple and efficient platform.
